Explication

Ce site servira à expliquer brièvement les tâches demandées par le projet SAE S1.03, à savoir l'installation et le déploiement d'une machine virtuelle.

Qu'est-ce qu'une machine virtuelle ?

Une machine virtuelle est un ordinateur logiciel qui fonctionne à l'intérieur d'une machine physique.

Il utilise des ressources matérielles partagées telles que le processeur, la mémoire et le stockage, tout en se comportant comme un ordinateur indépendant doté de son propre système d'exploitation.

Vidéo en anglais sur le thème des machines virtuelles

Comment déployer une machine virtuelle ?

Une machine virtuelle est déployée à l'aide d'un logiciel de virtualisation appelé hyperviseur.

Dans ce projet, VirtualBox est utilisé pour créer et configurer la machine virtuelle, installer un système d'exploitation Linux et exécuter des services tels qu'un serveur web.

Cette approche permet de tester, de développer et d'apprendre dans un environnement isolé et sécurisé.

Tâches SAE S1.03

L'objectif de ce projet est de découvrir les technologies de virtualisation en installant et en configurant un environnement de développement complet sur une machine virtuelle.

Ce projet a pour objectif de comprendre :

  • Quel système d'exploitation choisir pour cette tâche ?
  • Quel hyperviseur choisir pour cette tâche ?
  • Comment configurer une machine virtuelle localement
  • Comment configurer plusieurs comptes utilisateurs sur ladite machine virtuelle
  • Comment héberger un site web localement via une machine virtuelle
