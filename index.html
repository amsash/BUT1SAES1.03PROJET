Explaination

This site will serve to briefly explain the tasks requested by the SAE S1.03 project, namely being the installation and deployment of a virtual machine.

What is a virtual machine?

A virtual machine is a software-based computer that runs inside a physical machine.

It uses shared hardware resources such as the processor, memory, and storage, while behaving like an independent computer with its own operating system.

How is a virtual machine deployed?

A virtual machine is deployed using virtualization software called a hypervisor.

In this project, VirtualBox is used to create and configure the virtual machine, install a Linux operating system, and run services such as a web server.

This approach allows testing, development, and learning in an isolated and secure environment.

SAE S1.03 Tasks

The objective of this project is to discover virtualization technologies by installing and configuring a complete development environment on a virtual machine.

Through this project, the goal is to understand:

  • Which OS to choose for the task
  • Which hypervisor to choose for the task
  • How to set up a virtual machine locally
  • How to set up multiple user accounts on said virtual machine
  • How to host a website locally through the virtual machine
